Welcome to AtomixJs, a modern and efficient framework designed to simplify the process of creating robust web applications. By leveraging AtomixJs, you gain access to a suite of tools and functionalities that streamline the development of custom web components and state management, all while maintaining high performance and developer ergonomics.

## Quick Start

### Setting Up Your Project

To begin your journey with AtomixJs, you need to set up a new project and install the AtomixJs library. Choose your preferred package manager from the options below to add AtomixJs to your project:

<Tabs>
<TabItem label="npm">

```sh
npm install -D atomixjs
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm add -D atomixjs
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn add -D atomixjs
```

</TabItem>
<TabItem label="Bun">

```sh
bun add -D atomixjs
```

</TabItem>
</Tabs>

### First Steps

After installing AtomixJs, you're ready to start building. AtomixJs empowers you to create web components with enhanced lifecycle management and state handling capabilities, ensuring your applications are both scalable and maintainable.

Here's a quick overview of what you can achieve with AtomixJs:

- **Efficient Component Creation**: Utilize `makeAtomixComponent` to extend HTML elements with advanced features, including lifecycle management and state subscriptions.
- **Lifecycle and State Management**: Leverage built-in lifecycle hooks and state management utilities to create dynamic and responsive web applications.
- **Optimized for Performance**: AtomixJs is designed with performance in mind, ensuring your applications run smoothly across all devices.
