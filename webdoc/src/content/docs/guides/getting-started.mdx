AtomixJs is a simple and lightweight framework for building web applications. This guide will help you get started with AtomixJs.

## Quick Start

### Create a new project

<Tabs>
<TabItem label="npm">

```sh
npm install -D atomixjs
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm add -D atomixjs
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn add -D atomixjs
```

</TabItem>
<TabItem label="Bun">

```sh
bun add -D atomixjs
```

</TabItem>
